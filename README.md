# SaiKet-Systems-
Web Development Internship Tasks

____

This repository contains the deliverables completed during my Web Development internship at SaiKet Systems. The projects progress from structural UI layouts using HTML & CSS to dynamic, interactive applications powered by JavaScript and advanced DOM manipulation.

## Projects Overview

The internship was divided into 6 core progressive tasks, divided into two distinct phases.

---

#### Task 1: Online Portfolio Website
*   **Description:** Developed a professional personal portfolio website to showcase works, skills, and achievements.
*   **Tech Stack:** HTML5, CSS3, JavaScript (Optional)

#### Task 2: E-commerce Product Page
*   **Description:** Engineered a user-friendly product detail page layout featuring images, item details, star ratings, and an optimized "Add to Cart" functional area.
*   **Tech Stack:** HTML5, CSS3

#### Task 3: Simple Navigation Menu
*   **Description:** Created a streamlined webpage navigation system with optimized hyperlinks and modern CSS hover styles to ensure intuitive site usability.
*   **Tech Stack:** HTML5, CSS3

#### Task 4: HTML Forms & Validation
*   **Description:** Built a structured user-engagement form with various field types (text fields, radio buttons, checkboxes, dropdowns) coupled with clean input validation rules.
*   **Tech Stack:** HTML5, CSS3, JavaScript

#### Task 5: Dynamic Blogging Platform
*   **Description:** Programmed a frontend-driven content site enabling users to create, preview, edit, and publish custom blog articles dynamically.
*   **Tech Stack:** HTML5, CSS3, JavaScript (ES6+)

#### Task 6: Interactive Workspace Terminal
*   **Description:** Designed and engineered the *SaiKet Workspace Terminal*, a real-time dashboard featuring live task element injection, state counters, search indexing, and deep DOM structural manipulations.
*   **Tech Stack:** HTML5, CSS3, JavaScript (ES6+), DOM Engine Integration

---

## Key Skills Acquired

*   **Semantic Markup:** Writing production-grade, accessible HTML structures.
*   **Modern Styling:** Creating layout frameworks using responsive CSS.
*   **Logical Programming:** Building functional logic, event-listeners, and functional data routing in vanilla JavaScript.
*   **Dynamic UI Manipulation:** Leveraging DOM trees to interactively create, filter, and inject components based on real-time user action.

---

## Tech Stack Used

*   **Languages:** HTML5, CSS3, JavaScript (ES6+)
*   **Concepts:** DOM Manipulation, Client-side Form Validation, User Interface (UI) Design, Component Architecture
